# ASEP PASTE

Website seperti Pastebin, tanpa server. Teks disimpan di URL (hash). Bisa langsung di-host di GitHub Pages.

### Cara pakai
1. Buka `index.html`
2. Tempel teks
3. Klik **Simpan & buat link**
4. Salin link yang muncul dan bagikan

### Hosting di GitHub Pages
1. Buat repo baru bernama `ASEP-PASTE`
2. Upload file `index.html`, `style.css`, `script.js`, dan `README.md`
3. Aktifkan GitHub Pages dari **branch main**
4. Buka URL GitHub Pages, misalnya:  
   `https://username.github.io/ASEP-PASTE`

### Batasan
- Teks disimpan di URL, jangan gunakan untuk data sensitif
- Maksimum sekitar 30 KB teks
- Browser lama mungkin tidak mendukung fitur clipboard
